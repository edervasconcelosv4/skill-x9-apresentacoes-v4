# Análise de Referências - Lote 02

Foram analisados 10 decks enviados pelo usuário, todos em Google Slides, com leitura de outline e thumbnail visual do slide indicado em cada link.

## Resumo Executivo

Este lote confirma que a nova skill não deve tentar copiar apenas um template. Os melhores slides combinam uma assinatura V4 consistente com variações por contexto: capa cinematográfica para impacto, cards editáveis para estratégia, prints reais para prova e layouts de pipeline para processo.

## Tabela Comparativa

| # | Deck | Slide observado | Para que serve | O que copiar | Minha opinião |

| 1 | Movement Academia | Grade semanal de publicações | Organizar calendário editorial e rotina de stories | Tabela ampla com card lateral de rotina diária | Muito bom para conteúdo operacional. A tabela só funciona porque tem contraste, cor por pilar e rotina lateral. |

## O Que Isso Muda Na Skill

- Adicionar layouts oficiais: `cinematic_cover`, `evidence_diagnosis`, `pipeline_split_outcome`, `matrix_4x_highlight`, `executive_action_grid`, `market_scope_dashboard`, `weekly_calendar_plus_routine`.
- Permitir camada de fundo raster premium para capas e seções, mantendo textos, cards e dados editáveis.
- Tratar imagens de evidência como parte do argumento, não como decoração.
- Criar regras de densidade por tipo de slide: tabela aceita mais texto; capa quase nenhum; pipeline usa frases curtas; matriz exige textos compactos.

## Veredito

Sim, vale muito usar esses decks como referência. Eles mostram que o caminho certo não é "HTML bonito vs PPTX editável". É um sistema híbrido: fundo visual forte quando necessário, componentes editáveis por cima e uma biblioteca de layouts narrativos para cada tipo de raciocínio.
